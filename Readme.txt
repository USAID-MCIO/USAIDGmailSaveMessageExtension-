Gmail SaveAs Chrome Extension
----------------------------------

This Chrome Extension provides the ability to save Gmail message to the local system in the "eml" format. Extension deployed as an icon in the browser menu at the top. When a Gmail Inbox message is open in the chrome browser, user can click on the extension icon and save the email message to local system.

Eml is the standard format for e-mail messages recognized by Microsft Outlook Express as well as some other email programs.

For packaging, testing and deployment options, refer to https://developer.chrome.com/extensions
